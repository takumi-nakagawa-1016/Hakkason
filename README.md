# 環境構築
1. git clone git@github.com:takumi-nakagawa-1016/Hakkason.git
2. make setup
3. http://localhost:8080/login　にアクセスしてそのままログイン
4. create new milestoneを押すと一番上のマイルストーンを作成できる
5. 作成後は一覧を見れる
6. 一覧から＋ボタンを押すとそのマイルストーンに紐づくマイルストーンが作成できる
